What it is

A floating AI overlay, always at hand

Ghost AI is a transparent window that sits on top of any application. During meetings, calls, or interviews, it captures audio, transcribes it locally with Whisper, and generates contextual suggestions with Ollama — all without a single byte leaving your computer.

Features

Everything it does for you

A toolset built for live conversations: meetings, interviews, classes, and calls.

🎙️

Real-time transcription

Local Whisper transcribes your microphone, system audio (meetings and calls), or both simultaneously on separate channels.

🗣️

Speaker separation

"Speakers" mode labels who is talking: your mic is "You" and the other side of the call is split by voice into Speaker A / B / C — 100% on-device.

💡

Reply suggestions

Ollama generates contextual reply suggestions from the live transcription, with customizable prompts and an automatic mode.

📝

Summaries & translation

Summarize the conversation into bullet points with one click, or instantly translate the transcript to Brazilian Portuguese.

📸

Screen analysis

Capture the full screen or a cropped region and send it for AI analysis — great for code, presentations, and documents.

🪟

Floating overlay

Always-on-top transparent window, draggable, with adjustable opacity and global keyboard shortcuts (Cmd+Shift+G to show/hide).

🔄

Quick model switching

Switch between Ollama models (gemma, llama, mistral…) straight from the title bar without opening Settings.

💾

Conversation export

Save chats and transcripts to .txt files with AI-generated filenames. Nothing is persisted without your explicit action.

⌨️

Global shortcuts

Control everything from the keyboard even with the app in the background: capture screen, focus the input, show/hide the overlay.

Security & Privacy

Privacy isn't a feature. It's the architecture.

Ghost AI was designed on the principle that conversation data is too sensitive to leave your machine — and the implementation reflects that at every layer.

🛡️ Privacy guarantees

✓

Zero external calls during operation

All AI inference happens via Ollama at localhost:11434. The only exception is the initial model download (HuggingFace CDN), done once and then cached.

✓

No telemetry, analytics, or tracking

No cookies, no usage events, no identifiers. There is no data-collection code in the project — verifiable, since it's open source (MIT).

✓

No accounts or cloud sync

There is no sign-up, login, or synchronization. There's no server to leak your data, because there is no server.

✓

Ephemeral conversations by default

Chat history lives only in memory and is lost when the app closes — unless you explicitly export it to a .txt file.

🔐 Technical hardening (Electron)

✓

Context isolation

contextIsolation: true and nodeIntegration: false: the web UI has no access to Node.js. Only the minimal ghostAPI surface is exposed via contextBridge.

✓

Restrictive permission handler

The app grants only media, microphone, and screen — any other permission request is automatically denied.

✓

Clipboard via native IPC

Clipboard writes go through a controlled IPC channel (clipboard-write) instead of the web clipboard API, reducing the surface exposed to the renderer.

✓

Heavy models in the main process

Whisper and the voice-embedding model run in the main process, isolated from the UI — the renderer never touches the filesystem.

Getting started

Up and running in 4 steps

Install Ollama

brew install ollama and pull a model: ollama pull gemma4:latest (~3 GB). Alternatives: llama3.2, mistral, gemma3:12b.

Install Ghost AI

Download the DMG, drag it to Applications, and open it. On first launch (unsigned app), allow it under System Settings > Privacy & Security > "Open Anyway".

Grant the permissions

Microphone (transcribing your voice) and Screen Recording (system audio for meetings). The Whisper model (~118 MB) downloads automatically on first launch.

Use it in any conversation

Green dot = Ollama connected. Pick the audio source, hit record, and get live transcription + suggestions. Cmd+Shift+G shows/hides the overlay.
