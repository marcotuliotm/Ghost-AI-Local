O que é

Um overlay flutuante com IA, sempre à mão

O Ghost AI é uma janela transparente que fica sobre qualquer aplicativo. Durante reuniões, chamadas ou entrevistas, ele captura o áudio, transcreve localmente com Whisper e gera sugestões contextuais com Ollama — tudo sem que um único byte saia do seu computador.

Recursos

Tudo o que ele faz por você

Um conjunto de ferramentas pensado para conversas ao vivo: reuniões, entrevistas, aulas e chamadas.

🎙️

Transcrição em tempo real

Whisper local transcreve o microfone, o áudio do sistema (reuniões e chamadas) ou ambos simultaneamente, em canais separados.

🗣️

Separação de falantes

Modo "Speakers" identifica quem está falando: seu microfone é "Você" e o outro lado da chamada é separado por voz em Falante A / B / C — 100% on-device.

💡

Sugestões de resposta

O Ollama gera sugestões contextuais de resposta com base na transcrição ao vivo, com prompts personalizáveis e modo automático.

📝

Resumos e tradução

Resuma a conversa em tópicos com um clique ou traduza a transcrição para português brasileiro instantaneamente.

📸

Análise de tela

Capture a tela inteira ou uma região recortada e envie para análise da IA — útil para código, apresentações e documentos.

🪟

Overlay flutuante

Janela transparente sempre no topo, arrastável, com opacidade ajustável e atalhos globais de teclado (Cmd+Shift+G para mostrar/ocultar).

🔄

Troca rápida de modelo

Alterne entre modelos do Ollama (gemma, llama, mistral…) direto na barra de título, sem abrir as configurações.

💾

Exportação de conversas

Salve chats e transcrições em arquivos .txt com nomes gerados pela própria IA. Nada é persistido sem sua ação.

⌨️

Atalhos globais

Controle tudo pelo teclado mesmo com o app em segundo plano: capturar tela, focar o input, mostrar/ocultar overlay.

Arquitetura

Fluxo de dados: do áudio à sugestão

Todo o pipeline roda dentro da sua máquina. A única comunicação externa acontece uma única vez: o download inicial dos modelos.

Processamento local (Whisper / embeddings) Inferência LLM local (Ollama) Perímetro de privacidade

Arquitetura de processos do Electron

A separação entre processos segue as práticas recomendadas de segurança do Electron: a interface nunca tem acesso direto ao sistema.

Segurança & Privacidade

Privacidade não é um recurso. É a arquitetura.

O Ghost AI foi projetado a partir do princípio de que dados de conversas são sensíveis demais para sair da sua máquina — e a implementação reflete isso em cada camada.

🛡️ Garantias de privacidade

✓

Zero chamadas externas em operação

Toda a inferência de IA acontece via Ollama em localhost:11434. A única exceção é o download inicial dos modelos (HuggingFace CDN), feito uma única vez e depois cacheado.

✓

Sem telemetria, analytics ou tracking

Nenhum cookie, nenhum evento de uso, nenhum identificador. Não há código de coleta no projeto — verificável, pois é open source (MIT).

✓

Sem contas ou cloud sync

Não existe cadastro, login ou sincronização. Não há servidor para vazar seus dados, porque não há servidor.

✓

Conversas efêmeras por padrão

O histórico vive apenas em memória e é perdido ao fechar o app — a menos que você exporte explicitamente para um arquivo .txt.

🔐 Hardening técnico (Electron)

✓

Isolamento de contexto

contextIsolation: true e nodeIntegration: false: a interface web não tem acesso ao Node.js. Só a API mínima ghostAPI é exposta via contextBridge.

✓

Handler de permissões restritivo

O app concede apenas media, microphone e screen — qualquer outra permissão solicitada é negada automaticamente.

✓

Clipboard via IPC nativo

Escritas no clipboard passam por um canal IPC controlado (clipboard-write) em vez da API web de clipboard, reduzindo a superfície exposta ao renderer.

✓

Modelos pesados no processo principal

Whisper e o modelo de embeddings de voz rodam no main process, isolados da interface — o renderer nunca acessa o filesystem.

Como começar

Rodando em 4 passos

Instale o Ollama

brew install ollama e baixe um modelo: ollama pull gemma4:latest (~3 GB). Modelos alternativos: llama3.2, mistral, gemma3:12b.

Instale o Ghost AI

Baixe o DMG, arraste para Aplicativos e abra. Na primeira execução (app não assinado), autorize em Ajustes > Privacidade e Segurança > "Abrir mesmo assim".

Conceda as permissões

Microfone (transcrição da sua voz) e Gravação de Tela (áudio do sistema para reuniões). O modelo Whisper (~118 MB) baixa automaticamente na primeira execução.

Use em qualquer conversa

Ponto verde = Ollama conectado. Selecione a fonte de áudio, grave, e receba transcrição + sugestões em tempo real. Cmd+Shift+G mostra/oculta o overlay.

Stack técnica

Construído com tecnologias modernas

Requisito: macOS 13+ (Ventura) — a captura de áudio do sistema depende do ScreenCaptureKit da Apple.

⚛️ React 18 interface do overlay 🔷 TypeScript 5 tipagem em todo o código 🖥️ Electron 42 app nativo macOS ⚡ Vite build e dev server 🎨 Tailwind CSS estilo glassmorphism 🦙 Ollama LLM local (gemma, llama, mistral…) 🎙️ Whisper (ONNX) transcrição on-device (~118 MB) 🗣️ WavLM embeddings de voz para diarização 🤗 Transformers.js v4 runtime dos modelos 🧪 Vitest suíte de testes
